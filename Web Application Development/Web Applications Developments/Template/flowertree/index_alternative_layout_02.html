flowertree

welcome !

Hello, this is my third open source webdesign ! I call it flowertree because of the picture I used. It is 100% compliant with XHTML 1.0 Strict and does not contain any tables. I also tried to use relative metrics everywhere, so the layout should be very flexible (scaling, resizing, font sizes, etc.).


The source code (HTML and CSS) should be tidy and easy to to customize. I tested the design successfully in Mozilla Firefox 1.5, Opera 9.0 and Internet Explorer 6.0.


Feel free to use this template for any purpose you wish (personal, commercial, ...). If you publish a website that is based on flowertree, please email me your site's URI - that would be very nice (because I'm very curious)!
